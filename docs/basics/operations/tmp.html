1. PROGRAMMING, IN JAVA

  1. Which of the following is True about the Java architecture?
  1. Java code is compiled into bytecode by the Java Development Kit (JDK)
  2. Bytecode is an intermediate representation of the Java code that is executed by the Java Virtual Machine (JVM).
  3. Bytecode is executed by the Java Virtual Machine (JVM) to produce machine code that runs on the underlying hardware.
  4. JRE is the Java Runtime Environment that includes the JVM and other libraries.
  5. All of the above

Answer: D) All of the above

  1. Java is a:
  1. Compiled language: Java code is compiled into bytecode by the Java Development Kit (JDK) and executed by the Java Virtual Machine (JVM).
  2. Interpreted language: Java code is interpreted by the Java Development Kit (JDK) and executed by the Java Virtual Machine (JVM).
  3. Both compiled and interpreted language: Java code is compiled into bytecode by the Java Development Kit (JDK) and executed by the Java Virtual Machine (JVM).
  4. None of the above

Answer: A) Compiled language: Java code is compiled into bytecode by the Java Development Kit (JDK) and executed by the Java Virtual Machine (JVM).

  1. Java programming can only be done in Visual Studio Code.
  1. True
  2. False

Answer: B) False

  1. Java is a statically typed language, which means:
  1. The data type of a variable is determined at runtime and can change during execution.
  2. The data type of a variable is determined at compile time and cannot change.
  3. The data type of a variable is determined by the Java Virtual Machine (JVM) at runtime.
  4. The data type of a variable is determined by the Java Development Kit (JDK) at runtime.

Answer: B) The data type of a variable is determined at compile time.

2.1. Primitive Data Types

  1. Sort the following data types in Java from smallest to largest in terms of memory size:
  1. byte, short, int, long
  2. long, int, short, byte
  3. int, long, short, byte
  4. short, byte, long, int

Answer: A) byte, short, int, long

  1. double in Java is a:
  1. A numeric primitive data type that stores integer numbers bigger than supported by int.
  2. A numeric primitive data type that stores floating-point numbers more precisely than float.
  3. A numeric primitive data type that stores integer numbers bigger than supported by long.
  4. A numeric primitive data type that stores floating-point numbers less precisely than float.

Answer: B) A numeric primitive data type that stores floating-point numbers more precisely than float.

  1. Which of the following best describes char and String types in Java?
  1. char is a primitive data type that stores a single character, while String is a reference type that stores a sequence of characters.
  2. char is a reference type that stores a single character, while String is a primitive data type that stores a sequence of characters.
  3. char and String are both primitive data types that store single characters.
  4. char and String are both reference types that store sequences of characters.

Answer: A) char is a primitive data type that stores a single character, while String is a reference type that stores a sequence of characters.

  1. There is no primitive data type in Java for strings like str in Python.
  1. True
  2. False

Answer: A) True

  1. Converting an int i = 4; to a long type can be done in Java using:
  1. long l = (long) i;
  2. long l = long(i);
  3. long l = i.toLong();
  4. long l = i;

Answer: A) long l = (long) i;

  1. int and Interger in Java are:
  1. The same data type.
  2. Different data types.
  3. int is a primitive data type, and Integer is a wrapper class
  4. int is a wrapper class for the Integer primitive data type.

Answer: C) int is a primitive data type, and Integer is a wrapper class for the int primitive data type that allows it to be treated as an object.

  1. Every primitive data type in Java has a corresponding wrapper class that allows
  1. The primitive data type to be treated as an object.
  2. The primitive data type to be used to be used in loops.
  3. The primitive data type to be used as a return type in methods.
  4. The primitive data type to be used in conditional statements.

Answer: A) The primitive data type to be treated as an object.

  1. java int a = 10; long b = 20; float c = 30.0f; double d = 40.0; double result = a * b + c / d;

In the above code snippet, the data type of the result is determined by:

  1. The data type of the variable a because it comes first.
  2. The data type of the variable b because it is the largest integer.
  3. The data type of the variable c because it is a floating-point number.
  4. The data type of the variable d because it is most precise and the operand with the highest memory size.
